Любое поле должно содержать проверку на наличие данных. Если поле не заполнено, элемент не должен отображаться.

Если элемент содержит в себе несколько полей, например карточка с информацией, в которой хранится заголово и картинка, то делай предварительную проверку на наличие полей. В случае отсутствия одного из полей карточку отображать не нужно.

Для вывода элементов циклом используй while с проверкой на заполнненость поля.
